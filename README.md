# WouldYouRatherClone
A Clone of Would You Rather. 

A group-project for the students of fDevs.

Each member has a role they're filling in this project. We've split the current tasks into 4 roles.

Architect (Initial Creation, Assigning Responsibilities)
Back-End Developer (Database, MVC, WebAPI)
Front-End Developer (HTML5, CSS, UI)
Front-End Developer (JS, UX)
